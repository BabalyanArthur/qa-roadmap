# Bug Report: Картинки номер один и номер два не отображаются.

**ID:** BUG-002
**Severity:** High
**Priority:** Medium

**Environment:**
- Device: MacBook Air M4
- OS: macOS Sequoia
- Browser: Safari
- URL: https://the-internet.herokuapp.com/broken_images

**Steps to Reproduce:**
1. Открыть страницу Broken images.
2. Посмотреть загруженные изображения.

**Actual Result:**
Картинка номер один не отображается, отображает значек битого изображения.
Картинка номер два не отображается, отображает значек битого изображения.
Картинка номер три отображается

**Expected Result:**
Картинка номер один отображается.
Картинка номер два отображается.
Картинка номер три отображается.
